# CMD YouTube Video(s) Downloader 0.0.1




## Options
 * --url TEXT
 * --playlist TEXT
 * --help  shows help massege


## Commands
1)
``` bash
$ python run.py
```
2)
```bash
$ python run.py
Enter YouTube Video URL: 
playlist(y/n) : n
Folder Name : ./test/videos
```

Although the folder name is not needed it is required for downloading playlist
